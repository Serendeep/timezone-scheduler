# Timezone Scheduler

A timezone-aware event scheduler built with React and the Temporal API. Create events in one timezone and see them converted to others automatically.

Day 01 of 50 Days, 50 Projects.

## Features

- Schedule events with specific timezones
- View events in your local time or any other timezone
- Dark mode support (follows system preference)
- Persistent storage via localStorage

## Tech

- React 19
- TypeScript
- Tailwind CSS 4
- Temporal API (via polyfill)
- Vite

## Run locally

```bash
npm install
npm run dev
```
